# Pin Yin Display Script
A Swift Script that allows one to view the Pin Yin for Chinese characters selected by the cursor

# Usage

Compile the Swift script with `swiftc PinyinDisplay.swift -o PinYinDisplay -framework AppKit -framework ApplicationServices -framework Carbon`

Add to LaunchAgents with 'AddToLaunchAgents.sh'

Use `Control + Option + Command + P` to print the Pin Yin of highlighted chinese characters
